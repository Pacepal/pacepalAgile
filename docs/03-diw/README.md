# Diseno de Interfaces Web (DIW)

Documentacion del modulo DIW del proyecto PacePal.

Este modulo recoge el proceso de diseno de la interfaz de usuario de la aplicacion web.

Incluye investigacion, diseno visual, prototipos, accesibilidad y evaluacion de usabilidad.

---

## Estructura

### Sprint 1 - Diseno inicial

- Investigacion de tendencias y competencia
- Guia de estilos
- Wireframes
- Boceto final de la landing
