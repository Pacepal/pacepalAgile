## Requisitos obligatorios por sprint

### Sprint 1

- DIW: definición del proyecto multimedia, análisis de competencia, landing page responsive y vídeo de presentación.
- DWEC: formularios de registro y login con validaciones avanzadas, mensajes sin recarga y estructura preparada para backend.

### Sprint 2

- DIW: diseño completo de la aplicación, perfiles de usuario, WCAG 2.1 A, optimización WebP, apartado de usabilidad.
- DWEC: carrito interactivo, buscador AJAX, modularidad, tests funcionales y publicación en GitHub Pages.
- DWES: modelo relacional, API REST en PHP, sesiones, roles, documentación técnica y ejemplos de llamadas.

### Sprint 3

- DIW: integración de audio, vídeo y animaciones no intrusivas.
- DWEC: migración del escaparate a React conservando el comportamiento anterior.
- DAW: despliegue local con XAMPP, validación de Apache, BBDD, puertos, localhost y evidencias técnicas.
