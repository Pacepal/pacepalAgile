# Bitacora de trabajo

Esta carpeta recoge la bitacora y la documentacion de trabajo del proyecto PacePal. Los modulos no se duplican en la raiz de `docs/`: cada area vive aqui, organizada por modulo y sprint.

## Estructura

- `general/`: roadmap, reparto operativo y backlog vivo de tareas restantes.
- `requisitos/`: definicion del proyecto y requisitos normalizados.
- `agile/`: Scrum, tablero, historias de usuario, retrospectivas y feedback.
- `diw/`: diseno, interfaz, accesibilidad, responsive, identidad visual y multimedia.
- `dwec/`: cliente, validaciones, carrito, buscador, React y pruebas funcionales.
- `dwes/`: backend, API, sesiones, roles, modelo y Postman.
- `despliegue/`: instalacion local, XAMPP, rutas y evidencias de entorno.
- `sostenibilidad/`: sostenibilidad, documentos y materiales de concurso.

Las capturas y pruebas visuales se mantienen en `../evidencias/`. El material oficial del profesorado se mantiene en `../00-material/`. El entregable final se mantiene en `../09-entrega-final/`.
