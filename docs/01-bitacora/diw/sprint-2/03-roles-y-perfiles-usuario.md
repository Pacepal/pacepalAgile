# Gestión de usuarios y comportamiento por perfiles

La aplicación web **PacePal** contempla distintos tipos de usuarios con funcionalidades diferenciadas según su nivel de acceso. Esta separación permite organizar el uso de la plataforma y garantizar que cada tipo de usuario interactúe con el sistema únicamente dentro de los límites de su rol.

La definición de perfiles permite controlar el acceso a determinadas funcionalidades y facilita la gestión de la plataforma.

---

# Invitado (usuario anónimo)

El usuario invitado es cualquier visitante que accede a la aplicación sin haberse registrado ni iniciado sesión. Este tipo de usuario puede navegar por las secciones públicas de la web, consultar información sobre el proyecto y explorar el contenido disponible.

Entre sus funcionalidades principales se incluyen:

- Visualizar las rutas deportivas disponibles.
- Consultar actividades deportivas publicadas por otros usuarios.
- Acceder a la tienda y visualizar los productos disponibles.
- Navegar por las páginas informativas como la página de inicio o la sección **About**.

Sin embargo, el usuario invitado no puede realizar acciones que impliquen interacción directa con el sistema, como:

- Unirse a actividades.
- Crear nuevas actividades.
- Realizar compras en la tienda.

Para acceder a estas funcionalidades es necesario registrarse e iniciar sesión.

---

# Cliente (usuario registrado)

El usuario cliente corresponde a un usuario registrado en la plataforma que ha creado una cuenta y ha iniciado sesión en el sistema. Este perfil dispone de funcionalidades ampliadas que permiten una interacción activa con la comunidad.

Entre sus principales capacidades se encuentran:

- Crear nuevas actividades deportivas asociadas a rutas existentes.
- Unirse a actividades publicadas por otros usuarios.
- Consultar su historial de actividades en las que ha participado.
- Acceder a la tienda y realizar compras de productos deportivos.
- Gestionar la información de su perfil.

El perfil de cliente constituye el núcleo de la comunidad de **PacePal**, ya que permite la interacción entre usuarios y la organización de actividades deportivas en grupo.

---

# Administrador

El perfil de administrador tiene acceso completo a las funcionalidades de gestión de la aplicación. Este usuario se encarga de mantener el correcto funcionamiento de la plataforma y administrar los contenidos disponibles.

Sus responsabilidades principales incluyen:

- Gestionar las rutas deportivas disponibles en la plataforma.
- Administrar los productos disponibles en la tienda.
- Gestionar la información de los usuarios registrados.
- Supervisar las actividades publicadas por la comunidad.

Este perfil dispone de acceso al **panel de administración**, desde el cual se pueden realizar tareas de mantenimiento y control del sistema.
