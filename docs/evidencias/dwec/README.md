# Evidencias DWEC

Base de evidencias del modulo **DWEC** para PacePal.

## Sprint 1 - Formularios, validaciones y pruebas

Objetivo del sprint:

- implementar formularios funcionales y validar comportamiento en cliente con pruebas manuales y automatizadas

Alcance implementado y documentado:

- formulario de registro y login
- validaciones en cliente (incluyendo DNI)
- logica condicional de tarjeta
- pruebas funcionales manuales y automatizadas

- `registro-correcto.png`: registro correcto.

Cobertura funcional reflejada por las evidencias:

- flujo de alta de usuario con casos correctos e incorrectos
- flujo de autenticacion con respuesta esperada en error y exito
- validacion en vivo y reglas de formato en campos criticos
- logica condicional de tarjeta y validacion de datos asociados
