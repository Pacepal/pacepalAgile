# 00 · Inventario de fuentes revisadas

## Criterio de trabajo

Jerarquía aplicada para esta preparación documental:

1. Requisitos oficiales y rúbrica.
2. Documentación actual del repositorio por módulos y sprints.
3. Código, configuración, base de datos y tests como evidencia real.
4. Memorias antiguas (`pacepal.pdf` y `pacepal.odt`) solo como apoyo secundario.

Leyenda de fiabilidad usada en este inventario:

- `oficial`
- `requisito de sprint`
- `evidencia real`
- `documentación beta`
- `referencia`
- `pendiente`
