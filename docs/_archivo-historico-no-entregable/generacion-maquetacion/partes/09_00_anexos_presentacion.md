# 9. ANEXOS

Este bloque agrupa la documentación complementaria del proyecto PacePal. Cada anexo amplía
o detalla un aspecto concreto de la memoria principal, y puede consultarse de forma
independiente o como referencia durante la defensa.

Los anexos no repiten el cuerpo de la memoria: se enlazan desde él o lo complementan con
contenido técnico, visual o de proceso que haría demasiado denso el texto principal.
