## 4. Riesgos para el montaje en LibreOffice

| Riesgo                                  | Descripción                                                                             | Acción recomendada                                                                          |
| --------------------------------------- | --------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| Imágenes con espacios en el nombre      | `Diagrama de Gantt.png`, `Flujo loginSesionLogOut.png`, etc.                            | Al insertar en LibreOffice, usar el explorador de archivos para evitar errores de ruta      |
| Nombre de archivo con carácter especial | `Carrito_añadir_React.png` (ñ)                                                          | Verificar que el sistema de ficheros lo maneja correctamente; renombrar si da problemas     |
| Figura G5 con tres imágenes             | Se documenta como comparativa de tres resoluciones                                      | Insertar las tres imágenes seguidas con el mismo pie de figura o como una composición       |
| Referencias cruzadas en PDF             | Los anclas `<a id="...">` del documento fuente no funcionan en LibreOffice directamente | Usar marcadores de Word/LibreOffice para que las referencias del índice funcionen en el PDF |
