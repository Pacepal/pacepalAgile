## 5) Decisiones tomadas

- una sola app intermodular  
  no se hacen proyectos separados por asignatura
- rutas fijas en base de datos  
  no hay alta de rutas por usuario
- actividad es lo central  
  crear, unirse, salir
- nivel y ritmo van por categorías simples  
  nivel: novato, intermedio, avanzado  
  ritmo: suave, medio, rápido
- seguridad se plantea desde inicio  
  DNI en registro  
  reportes y avisos con moderación admin
- tienda entra como parte de sostenibilidad y modelo de negocio  
  catálogo, carrito, pedido simple

## 6) Problemas o bloqueos
